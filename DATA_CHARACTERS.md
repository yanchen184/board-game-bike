# 🚴‍♂️ 角色系統詳細數據表

## 角色屬性說明

### 基礎屬性定義
- **速度 (Speed)**：平路巡航速度，影響基礎移動速度
- **耐力 (Stamina)**：體力上限和恢復速度
- **爬坡 (Climbing)**：上坡時的效率和速度保持
- **衝刺 (Sprint)**：短時間爆發力和加速能力
- **恢復 (Recovery)**：體力恢復速度和效率
- **團隊配合 (Teamwork)**：隊形效果加成和輪替效率

### 數值範圍
- 0-100：主要屬性值
- 每10點差距代表明顯的能力差異
- 60為平均水準，80為優秀，90+為頂尖

## 平衡性說明

### 設計理念
1. **無絕對強者**：每個角色都有明確的優勢和劣勢
2. **組合大於個體**：團隊配合比單一強力角色更重要
3. **策略多樣性**：至少存在5種以上的可行策略
4. **風險與回報**：強力能力伴隨明顯代價

### 數值平衡公式
```
角色總值 = Σ(六項屬性) ± 特殊能力價值
標準總值 = 440 ± 20
```

所有角色的總體價值保持在420-460之間，確保平衡性。
