# 🚲 裝備系統詳細數據表

## 裝備系統概述

### 裝備分類
1. **車架 (Frame)** - 影響重量、空氣動力、耐久度
2. **輪組 (Wheelset)** - 影響空氣動力、重量、穩定性
3. **變速系統 (Drivetrain)** - 影響變速精準度、效率、耐用性
4. **配件 (Accessories)** - 提供額外功能和加成

### 裝備評級系統
- ⭐ 入門級：便宜但性能有限

## 七、裝備維護與故障系統

### 故障機率計算

基礎故障率 = (100 - 耐久度) / 100 * 路程係數 * 天氣係數

| 裝備狀態 | 耐久度 | 故障機率/100km | 維修時間 |
|----------|--------|-----------------|----------|
| 全新 | 100% | 0.5% | - |
| 良好 | 80-99% | 1-2% | 5分鐘 |
| 一般 | 60-79% | 3-5% | 10分鐘 |
| 磨損 | 40-59% | 6-10% | 15分鐘 |
| 嚴重磨損 | 20-39% | 11-20% | 20分鐘 |
| 瀕臨報廢 | <20% | >20% | 30分鐘 |

### 維護建議
1. **預防性維護**：每100km檢查一次

3. **緊急維修**：消耗備用零件
4. **團隊互助**：輔助手可加速維修

## 八、裝備數值平衡公式

### 性能計算公式
```javascript
// 總性能分數
performanceScore =
  (frameScore * 0.35) +
  (wheelScore * 0.35) +
  (drivetrainScore * 0.25) +
  (accessoryScore * 0.05)

// 性價比計算
valueRatio = performanceScore / (totalCost / 1000)

// 重量懲罰
weightPenalty = Math.max(0, (totalWeight - 6800) * 0.001)

// 最終速度加成
speedBonus = performanceScore * (1 - weightPenalty) * setBonus
```

### 平衡性保障
- 最貴裝備性能上限：普通裝備的150%
- 最便宜裝備性能下限：頂級裝備的60%
- 套裝加成上限：25%
- 單項極限優化代價：其他屬性至少降低20%
