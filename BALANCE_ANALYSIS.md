# 一日北高挑戰 - 遊戲平衡性分析報告

## **BALANCE ASSESSMENT**: **Needs Tuning**

當前遊戲存在多個平衡性問題，包括角色價值不均、優勢策略過於明顯、以及某些機制缺乏深度。

## **KEY FINDINGS**

1. **破風手（Domestique）性價比過高** - 只要 $1100 但提供團隊體力消耗 -15% 的強力效果
2. **火車隊形（Train）優勢過大** - 25% 風阻減少配合破風手幾乎成為唯一最優解
3. **爬坡路段影響不足** - 只有 80km 爬坡（21%），爬坡手價值偏低
4. **裝備選擇缺乏意義** - 預算限制下只有一種明顯最優配置
5. **體力系統過於寬鬆** - 24小時完成380公里難度太低

## **DETAILED ANALYSIS**

### 1. 角色平衡分析

#### 爬坡手 (Climber) - **Underpowered**
- **成本**: $1200
- **特殊能力**: 山路加成 +20%
- **問題分析**:
  - 380km 路線中只有 80km 爬坡路段（seg_3: 40km, seg_5: 40km）
  - 爬坡路段佔比僅 21%，特殊能力發揮空間有限
  - 爬坡時速度計算: `terrainMultiplier = 0.6 + (climbing/100) * 0.3`
  - 95 climbing 給予 88.5% 速度，但只在 21% 路程有效
  - 實際價值遠低於成本

#### 衝刺手 (Sprinter) - **Balanced but Risky**
- **成本**: $1300
- **特殊能力**: 平路加成 +25%
- **問題分析**:
  - 300km 平路（79% 路程）可發揮優勢
  - 但體力只有 60，在長距離賽事中風險極高
  - 速度優勢（100 speed）可能無法持續發揮
  - 高風險高回報的設計，但回報不足以彌補風險

#### 破風手 (Domestique) - **Overpowered**
- **成本**: $1100（最便宜）
- **特殊能力**: 團隊體力消耗 -15%
- **問題分析**:
  - 全隊受益的被動效果，價值遠超其他角色
  - 90 體力 + 85 恢復力 + 100 團隊合作數值優秀
  - 成本最低但提供最高團隊價值
  - 幾乎是必選角色，破壞了策略多樣性

#### 全能選手 (All-Rounder) - **Underpowered**
- **成本**: $1000
- **特殊能力**: 無實際數值加成
- **問題分析**:
  - 所有屬性 75 看似平衡，但缺乏特色
  - 沒有任何情況下優於專門角色
  - "適應各種路況"沒有實際遊戲機制支撐

### 2. 隊形系統分析

#### 當前隊形效果
- **單線 (Single)**: 20% 風阻減少
- **雙線 (Double)**: 15% 風阻減少
- **火車 (Train)**: 25% 風阻減少

#### 問題
1. **火車隊形過強**: 最高風阻減少且沒有明顯缺點
2. **缺乏權衡**: 沒有機制讓玩家需要切換隊形
3. **單調性**: 永遠選擇火車隊形是最優解

### 3. 裝備系統分析

#### 預算分配問題
- 總預算 $5000
- 4人團隊最低成本: $4000（4個全能選手）
- 剩餘裝備預算: $1000

#### 裝備選擇缺乏深度
- 車架選擇被預算限制（只能選鋼管 $800）
- 輪組和變速系統選擇空間極小
- 高端裝備（碳纖維）完全無法負擔

### 4. 遊戲難度分析

#### 時間限制過於寬鬆
- 24小時完成380km = 平均速度 15.8 km/h
- 遊戲中正常速度範圍: 5-45 km/h
- 計算顯示即使保守配置也能在12-15小時完成

#### 體力系統問題
- 基礎消耗率: 0.02%/秒 = 72%/小時
- 領騎者 1.5x 消耗 = 108%/小時
- 但有自動輪換機制，實際消耗大幅降低
- 補給站恢復（最高30%）過於慷慨

## **POTENTIAL ISSUES**

### 1. **優勢策略過於明顯** [Critical]
   - **Impact**: 遊戲缺乏重玩價值，策略單一
   - **Evidence**:
     - 最優組合: 3個破風手 + 1個全能選手 = $4300
     - 永遠使用火車隊形（25%風阻減少）
     - 團隊體力消耗降低45%（3x15%）
     - 剩餘$700可買基礎裝備

### 2. **路線設計單調** [Major]
   - **Impact**: 降低角色選擇多樣性
   - **Evidence**:
     - 79%平路讓爬坡手價值極低
     - 沒有純下坡路段
     - 難度分布不均（最高難度5只出現一次）

### 3. **隨機事件影響過小** [Minor]
   - **Impact**: 策略規劃過於確定，缺乏應變需求
   - **Evidence**:
     - 機械故障最多延遲4分鐘
     - 天氣影響只有±15%速度
     - 補給站選擇差異不大

## **RECOMMENDATIONS**

### 1. **重新定價角色成本**
   - **Rationale**: 平衡性價比，增加選擇困難度
   - **Implementation**:
     ```javascript
     破風手: $1100 → $1400 (提高29%)
     爬坡手: $1200 → $1000 (降低17%)
     衝刺手: $1300 → $1200 (降低8%)
     全能選手: $1000 → $900 (降低10%)
     ```

### 2. **增加隊形切換機制**
   - **Rationale**: 不同路段需要不同隊形
   - **Implementation**:
     - 單線: 爬坡+15%速度，平路風阻-20%
     - 雙線: 平衡型，風阻-15%，側風穩定性+20%
     - 火車: 平路風阻-25%，但爬坡-10%速度，轉彎-15%速度

### 3. **改進路線設計**
   - **Rationale**: 增加地形多樣性，提升專門角色價值
   - **Implementation**:
     - 增加爬坡路段至120km (32%)
     - 新增下坡路段60km
     - 加入技術路段（彎道、窄路）需要控制技能

### 4. **強化體力消耗機制**
   - **Rationale**: 增加資源管理深度
   - **Implementation**:
     ```javascript
     基礎消耗: 0.02 → 0.03%/秒
     領騎者倍率: 1.5x → 2.0x
     低體力懲罰: <30%時速度-20%
     極限模式: <10%時速度-50%，可能DNF
     ```

### 5. **裝備系統分層設計**
   - **Rationale**: 讓不同預算策略都可行
   - **Implementation**:
     - 新增中階裝備選項（$1000-2000區間）
     - 裝備組合加成（同品牌額外+5%效果）
     - 裝備耐久度影響（低價裝備故障率更高）

### 6. **動態事件系統**
   - **Rationale**: 增加遊戲變化性和挑戰
   - **Implementation**:
     - 事件強度隨遊戲進度增加
     - 連鎖事件（雨天增加爆胎機率）
     - 團隊事件（隊員受傷需要護送）

## **PLAYTESTING PRIORITIES**

### 高優先級測試場景

1. **極限預算測試**
   - 最低預算配置（4x全能選手）完成時間
   - 最高預算效率配置性能測試
   - 確認是否存在預算甜蜜點

2. **角色組合測試**
   - 單一類型x4（如4個破風手）
   - 平衡組合（每種一個）
   - 極端組合（3衝刺手+1破風手）

3. **體力耗盡測試**
   - 不停靠補給站能否完成
   - 最少補給次數測試
   - 極限速度推進測試

4. **隊形切換時機**
   - 測試不同地形最優隊形
   - 評估切換成本vs收益
   - 動態隊形策略可行性

5. **隨機事件連鎖**
   - 最壞情況模擬（所有負面事件）
   - 事件恢復時間測試
   - 事件對最終成績影響權重

## **深度與策略性改進建議**

### 1. 新增技能系統
- 每個角色可選擇1-2個主動技能
- 技能有冷卻時間和體力消耗
- 例如：爆發衝刺、團隊激勵、緊急修理

### 2. 天氣預報系統
- 賽前可看3小時天氣預報
- 玩家需據此調整策略
- 天氣影響裝備選擇

### 3. 疲勞累積機制
- 長期疲勞vs短期體力
- 休息品質影響恢復
- 過度疲勞增加受傷風險

### 4. 賽段積分系統
- 不同路段設置積分點
- 鼓勵多樣化策略
- 積分影響最終評分

### 5. 對手AI系統
- 加入AI車隊競爭
- 不同難度AI有不同策略
- 可以觀察和學習AI策略

## **總結**

當前遊戲框架良好，主題吸引人，但平衡性需要顯著調整。主要問題在於：
1. 破風手過強導致策略單一化
2. 路線設計偏向平路降低了角色多樣性
3. 裝備系統受預算限制缺乏選擇空間
4. 難度曲線過於平緩，缺乏挑戰性

通過實施建議的調整，特別是角色重新定價、隊形系統改進、和體力消耗強化，可以顯著提升遊戲的策略深度和重玩價值。建議優先進行角色平衡調整，然後逐步實施其他系統改進。
